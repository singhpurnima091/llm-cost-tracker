# LLM Cost Tracker 🚀

A Python CLI tool to track and monitor LLM token usage and cost.

## Features
- Track token usage
- Calculate cost
- Store logs in JSON
- Generate reports

## Installation

```bash
pip install -e .
## 🚀 Usage

### Track usage
```bash
llm-tracker --tokens 2000
Generate Report
llm-tracker --report
